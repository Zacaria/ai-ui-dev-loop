# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please do not open a public issue. Use one of
the following channels:

- GitHub Security Advisories (preferred): open a private advisory in this repo.
- If advisories are not available, contact the maintainer via GitHub: @Zacaria

You should receive a response within 7 days. If you do not hear back, please
follow up with a reminder.

## Supported Versions

This is a demo project. Security fixes are applied on a best-effort basis to
the default branch.
